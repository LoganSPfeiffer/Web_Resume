# Design Decisions

## Theme
- Dark theme: bg-gray-950 base, text-gray-100 primary text
- Accent: blue-400/blue-500 for headings, links, highlights
- Secondary accent: emerald or teal for contrast elements
- Cards: bg-gray-900 with border-gray-800

## Layout
- Max width: max-w-4xl (896px) centered
- Section padding: py-16 or py-20
- Section dividers: border-t border-gray-800
- Hero: full-width dark gradient, extra tall (min-h-screen or py-32)

## Typography
- Name (h1): text-5xl or text-6xl font-bold
- Section headings (h2): text-2xl font-bold uppercase tracking-wider text-blue-400
- Body: text-gray-300, text-base
- Muted/dates: text-gray-500, text-sm

## Interactive Features
- Sticky nav: fixed top, bg-gray-950/80 backdrop-blur
- Smooth scroll: scroll-behavior smooth on html
- Hover cards: transition-all duration-300 hover:scale-105 or hover:border-blue-500
- Skill bars: width animation via CSS transition
- Section fade-in: opacity transition on scroll

## Inspiration
- Dark portfolio sites with bold color sections
- Clean card-based layouts for projects
- Timeline layout for experience
- Badge/pill style for skills and certifications
